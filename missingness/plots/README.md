# Missingness Diagnostic Plots

This folder contains retained PNG/PDF figure pairs generated by the missingness scripts.

The retained plot set uses `10,000` bootstrap iterations, `10,000` M5 Monte Carlo iterations, deterministic
master seed `20260522`, and the shared `../../analysis/src/plot_style.py` style at `600` DPI. Multi-panel
plots use shared axes within each figure to avoid visual scale confusion.

## Files

| file | analysis block |
|---|---|
| `M1_concentration_vs_missingness.*` | Daily network mean PM2.5 vs daily missingness fraction. |
| `M2_sd_vs_missingness.*` | Cross-sensor daily SD vs daily missingness fraction. |
| `M2_cv_vs_missingness.*` | Cross-sensor daily coefficient of variation vs daily missingness fraction. |
| `M3_seasonal_missingness.*` | Seasonal missingness summaries. |
| `M4_gap_length_histograms.*` | Missing-gap duration distributions. |
| `M5_period_mdape_curves.*` | Period-level Monte Carlo MdAPE sensitivity curves. |
| `M5_daily_fraction_curves.*` | Fraction of days meeting daily MdAPE thresholds under completeness scenarios. |
| `M6_morans_i_sensitivity.*` | Moran's I completeness sensitivity by network and distance band. |
| `PM25_three_city_daily_sensor_timeseries_common_full_scale.*` | Daily sensor-level PM2.5 traces for Dhaka, Lucknow, and Chicago on a full common y-scale; mean is black and median is yellow. |
| `PM25_three_city_daily_sensor_timeseries_common_p995_zoom.*` | Same daily sensor-level traces on a common 99.5th-percentile zoom scale for readability. |
| `three_city_daily_pm25_timeseries_summary.csv` | Input counts and y-scale metadata for the three-city daily time-series plots. |

## Reproduction

Run from the repository root:

```bash
python missingness/scripts/run_missingness_analysis.py
python missingness/scripts/plot_three_city_pm25_timeseries.py
```
